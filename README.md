# Iris Classification Project

Python course project - classifying Iris flowers and visualizing results.

## Files
All project files are in the `iris-classification/` folder.

## How to Run
1. Install requirements: `pip install scikit-learn matplotlib numpy seaborn`
2. Run the Python files in order.

## What's Included
- 4 completed tasks with visualizations
- Python code for each task
- Generated images
- PDF report

## Results
See the images in the `iris-classification/images/` folder.
